# HandPose with Arduino

This examples demonstrates how to use ml5.js HandPose with Arduino using Webserial.

When the users fingers are up the LEDs will turn on and when the users fingers are down the LEDs will turn off. 

Code for the p5.js sketch can be found [here](https://editor.p5js.org/gracywhelihan/sketches/zjk6m6_4h)
